* sudo apt install openssh-server:
sudo - Execute as administrator.
apt install - Search for and install Ubuntu software packages.
* ip a:
Confirms network interfaces on the system and their current status (UP or DOWN).
* ping -c *number* *example.com*
-c - Count; stops ping command after *number* of times.





\# Day 2 (2/4/26):

* sudo chmod 600:
chmod - Change access permissions. (CHange MODe).
600 - Octal number mode for enabling only a use to read and write a file. Denoted as -rw-------.



* ls -l /etc/netplan/
ls - List information about files.
-l - Long listing format; in the case of /etc/netplan/, it lists their access permissions.
