# Home-Lab-Network
Home Lab Network project, created using VirtualBox to design my own home network.
Goal - To create a working internal network that utilises VMs, routing, and other network tools and capabilities.
Tools - VirtualBox, pfSense, Wireshark, Ping/Traceroute.

Project steps:
1) Basic network build - Create a small internal network of two VMs with static IP addresses.
2) Network control - Introduce a router through pfSense.
3) Automate networking - Enable DHCP and DNS services.
4) Firewall and segmentation - Segment the network and add firewall rules.
5) Deploying services - Implement web server functionality.
6) Traffic analysis - Utilise WireShark to capture packets and monitor traffic.
7) Troubleshooting - Intentionally break and misconfigure systems, and work to fix them.
8) Documentation - Finalise documents.
9) Additional optional upgrades - Add VPN, VLAN, cloud connnectivity, IaC functionality.
